🔑 投稿の初期設定（初回のみ）

投稿はGitHubリポジトリに直接保存されます。Fine-grained personal access token を作成して貼り付けてください。

1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
2. Repository access: jawameg のみを選択
3. Permissions → Contents: Read and write
4. 生成されたトークン（github_pat_…）を下に貼り付け

トークンはこの端末のブラウザ内にのみ保存されます。
