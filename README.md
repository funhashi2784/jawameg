# 🏮 青森ねぶたマップ

青森県内のねぶた設置スポットをまとめたインタラクティブマップアプリです。

## 使い方

`index.html` をブラウザで開くか、GitHub Pages / Netlify でホスティングしてください。

## 構成

```
index.html   # メインのマップアプリ
imgs/        # 各スポットの写真（49枚）
```

## デプロイ

### GitHub Pages
1. このフォルダの内容をGitHubリポジトリにpush
2. Settings → Pages → Source: `main` ブランチ / `/ (root)` を選択
3. 発行されたURLでアクセス可能

### Netlify Drop
1. https://app.netlify.com/drop を開く
2. このフォルダをドラッグ&ドロップ
